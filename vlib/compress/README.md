## Description

`compress` is a namespace for (multiple) compression algorithms supported by V.

At the moment, the following compression algorithms are implemented:

- `compress.bzip2`
- `compress.brotli`
- `compress.deflate`
- `compress.gzip`
- `compress.lz` supporting the following variations:
  - lz4
  - lz77
  - lz78
  - lzjb
  - lzma
  - lzma2
  - lzss
  - lzw
- `compress.snappy`
- `compress.zlib`
- `compress.zstd`
