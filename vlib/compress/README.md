## Description

`compress` is a namespace for (multiple) compression algorithms supported by V.

At the moment, the following compression algorithms are implemented:

- `compress.bzip2`
- `compress.deflate`
- `compress.gzip`
- `compress.lz4`
- `compress.lz77`
- `compress.lz78`
- `compress.lzjb`
- `compress.lzma`
- `compress.lzma2`
- `compress.lzss`
- `compress.lzw`
- `compress.snappy`
- `compress.zlib`
- `compress.zstd`
